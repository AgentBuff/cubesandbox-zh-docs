Cube SandboxEmpowering your AI Agents.

极速启动、高并发、安全且轻量化的 AI Agent 沙箱。

快速开始

⚡ 极速启动

资源池化预置 + 快照克隆，跳过所有冷启动开销。创建沙箱比一次眨眼都快。

🔒 硬件级隔离

每个沙箱配备独立操作系统内核，运行在专属 MicroVM 中。

🔌 E2B 生态兼容

完全兼容 E2B SDK 接口，替换一个环境变量即可从 E2B 云无缝切换，零业务代码改动。

📦 高密度部署

单沙箱额外开销仅 MB 级，通过内核共享与写时复制（CoW），单机可运行数千个实例。

🛡️ 网络安全

基于 eBPF 的 CubeVS 在内核态实现严格的沙箱间网络隔离，支持细粒度出站流量过滤策略。

📸 灵活的状态管理

支持 Checkpoint 保存、任意状态回滚与快速分叉，为并行开发和多版本测试提供前所未有的灵活性。（即将开源）

开始使用 ​

  • 快速开始 — 几分钟内从零到运行沙箱
  • 本地构建部署 — 从源码构建并在单机上部署
  • 多机集群部署 — 扩展到多节点集群
  • 架构概览 — 了解系统设计与核心组件

场景示例 ​

SDK 示例与端到端场景：

  • 示例项目 — 代码执行、浏览器自动化、OpenClaw 集成、RL 训练等场景
  • 仓库示例合集 — GitHub 上的完整示例
